# Portfolio Website

My personal portfolio website I built using Flask and Tailwind CSS.


## Features
- Responsive design with Tailwind CSS
- Projects showcase
- Contact form with email functionality
- Skills page

## Tech Stack
- **Backend**: Flask, Python
- **Frontend**: HTML, CSS, JavaScript, Tailwind CSS
- **Database**: SQLite
